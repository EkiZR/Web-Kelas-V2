# Web Kelas
Web Kelas adalah platform yang bertujuan untuk memberikan informasi mengenai kelas kami

Teknologi yang digunakan diantaranya: 
- React JS
- MUI
- Tailwind CSS
- Slick JS
- Firebase

Apa yang ada di web ini :
- Text Anonim
- Gallery
- Structure And Schedule
